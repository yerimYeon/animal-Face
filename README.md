### **프로젝트 제작하게 된 이유**

><br/> 친구들과 함께 서로의 동물상을 추측해보던 경험이 생각나서 내가 만든 프로그램을 통해 자신의 동물상이 무엇인지 알아보면 재밌을 것같아 만들게 된 프로그램입니다. 
>
>이 프로그램은 자신의 사진을 업로드하고 토끼, 공룡, 곰, 강아지, 고양이 총 다섯종류의 동물에서 맞는 수치가 나타나도록 되어있습니다.
